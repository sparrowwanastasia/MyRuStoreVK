MyRuStore
Мини-витрина приложений в стиле RuStore (Jetpack Compose, Kotlin)

- Экран онбординга
- Каталог приложений с категориями
- Экран деталей
- Навигация через Navigation Compose
- Локальные данные и сохранение состояния

Как собрать

git clone https://github.com/USERNAME/MyRuStore.git
cd MyRuStore
./gradlew assembleDebug
